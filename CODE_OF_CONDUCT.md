# Code of Conduct

## Our standard

This project aims to be a friendly, professional space. Be respectful, assume good intent, and keep the
focus on the work. Harassment, personal attacks, discrimination, and other unwelcoming behavior are not
tolerated — in issues, pull requests, discussions, or any other project space.

## Scope

This applies to everyone participating in the project's spaces, and to anyone representing the project
in public.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers via a
**GitHub private security advisory** (the **Security** tab → **Report a vulnerability**) — it's the
project's private contact channel, and reports are handled confidentially.

This is a small, best-effort open-source project, so we'll review reports and respond as we're able —
there's no guaranteed timeline. Maintainers may remove, edit, or reject contributions, and may warn or
remove anyone whose behavior they judge inappropriate.

This Code of Conduct is adapted in spirit from the
[Contributor Covenant](https://www.contributor-covenant.org).
